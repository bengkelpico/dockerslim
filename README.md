# MKCERT Local Dev

Create local https cert in bash/shell

`$ mkcert -cert-file pico-cert.pem -key-file pico-key.pem pico.dev "*.pico.dev" php.dev "*.php.dev" local.test pico test "*.pico.test" localhost 127.0.0.1 ::1`
